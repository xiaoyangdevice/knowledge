# 高血压临床诊疗指南

## 一、概述

高血压是以体循环动脉血压持续升高为主要特征的慢性疾病，是心脑血管疾病最重要的危险因素。我国高血压患病率逐年上升，目前约有2.45亿高血压患者。

## 二、诊断标准

### 2.1 血压分级

| 分级 | 收缩压(mmHg) | 舒张压(mmHg) |
|------|-------------|-------------|
| 正常血压 | <120 | <80 |
| 正常高值 | 120-139 | 80-89 |
| 1级高血压（轻度） | 140-159 | 90-99 |
| 2级高血压（中度） | 160-179 | 100-109 |
| 3级高血压（重度） | ≥180 | ≥110 |

### 2.2 诊断要点

1. 非同日3次诊室血压测量，收缩压≥140mmHg和/或舒张压≥90mmHg
2. 24小时动态血压监测（ABPM）：24h平均血压≥130/80mmHg
3. 家庭自测血压：≥135/85mmHg

## 三、治疗原则

### 3.1 非药物治疗

- **饮食调整**：减少钠盐摄入（每日<6g），增加钾摄入
- **控制体重**：BMI控制在18.5-23.9 kg/m²
- **规律运动**：每周≥5天，每次30分钟的中等强度有氧运动
- **限制饮酒**：男性每日酒精量<25g，女性<15g
- **戒烟**：完全戒烟，避免被动吸烟
- **心理平衡**：减轻精神压力，保持心态平和

### 3.2 药物治疗

#### 一线降压药物

1. **钙通道阻滞剂（CCB）**：氨氯地平、硝苯地平控释片
2. **血管紧张素转换酶抑制剂（ACEI）**：依那普利、贝那普利
3. **血管紧张素II受体拮抗剂（ARB）**：缬沙坦、厄贝沙坦
4. **利尿剂**：氢氯噻嗪、吲达帕胺
5. **β受体阻滞剂**：美托洛尔、比索洛尔

#### 联合用药方案

- ACEI/ARB + CCB
- ACEI/ARB + 利尿剂
- CCB + 利尿剂
- CCB + β受体阻滞剂

## 四、目标血压

- 一般高血压患者：<140/90mmHg
- 65-79岁老年人：<150/90mmHg，如耐受可降至<140/90mmHg
- ≥80岁老年人：<150/90mmHg
- 合并糖尿病、慢性肾病：<130/80mmHg

## 五、随访管理

- 血压达标且稳定者：每1-3个月随访一次
- 血压未达标或不稳定者：每2-4周随访一次
- 每年至少进行一次全面评估，包括靶器官损害和心血管危险因素
