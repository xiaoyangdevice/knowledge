# 中药方剂配伍指南


## 一、方剂配伍原则——君臣佐使


方剂是在辨证审因确定治法后，按照组方原则，选择适当药物并酌定用量，按一定剂型组合而成的处方。方剂的组成遵循"君臣佐使"的配伍原则：


**君药**：针对主病或主证起主要治疗作用的药物，在方中用量较大、药力较强，是不可缺少的核心药物。


**臣药**：辅助君药加强治疗主病或主证的药物；或针对兼病或兼证起治疗作用的药物。


**佐药**：①佐助药：配合君臣药加强治疗或治疗次要症状；②佐制药：减轻或消除君臣药的毒性或峻烈之性；③反佐药：与君药性味相反而又能起相成作用的药物。


**使药**：①引经药：引导诸药直达病所；②调和药：调和方中诸药，如甘草在许多方中起调和作用。


## 二、方剂的变化形式


1. **药味加减变化**：在基本方的基础上，随证加减药物。如桂枝汤加厚朴、杏仁则成桂枝加厚朴杏子汤，增强降气平喘之功。


2. **药量增减变化**：方中药物相同，但因药量比例不同而功效各异。如桂枝汤与桂枝加桂汤，后者增桂枝用量至五两以平冲降逆。小承气汤与厚朴三物汤药味相同，但主次剂量不同，功效有别。


3. **剂型更换变化**：同一方剂因剂型不同，作用强弱有别。如理中丸改为理中汤则药力较猛。抵当汤改为抵当丸则药力较缓。


## 三、常用药对配伍


药对是临床常用的两味药物固定搭配，具有协同增效或制约减毒的作用：


**黄芪-当归**：补气生血。黄芪大补脾肺之气，当归补血活血，两药合用气旺血生，为当归补血汤之核心组合。


**柴胡-黄芩**：和解少阳。柴胡疏散退热、疏肝解郁，黄芩清热燥湿泻火，两药合用一散一清，为小柴胡汤之核心组合。


**麻黄-桂枝**：发汗解表。麻黄开腠理、发汗散寒，桂枝温经通阳、助麻黄发汗，两药合用增强发汗解表之力，为麻黄汤之君臣配伍。


**黄连-肉桂**：交通心肾。黄连清心泻火，肉桂温肾助阳引火归元，两药寒热互用、心肾相交，为交泰丸之经典药对。


**半夏-黄连**：辛开苦降。半夏辛温燥湿化痰、降逆止呕，黄连苦寒清热燥湿，两药合用调理脾胃、消痞散结，为半夏泻心汤之核心药对。


**附子-干姜**：回阳救逆。附子回阳救逆、补火助阳，干姜温中散寒、回阳通脉，两药合用相须为用，为四逆汤之君臣组合。


**白芍-甘草**：缓急止痛。白芍养血敛阴、柔肝止痛，甘草缓急和中，两药合用酸甘化阴、缓急止痛，为芍药甘草汤之全方。


**大黄-芒硝**：峻下热结。大黄泻热通便、荡涤肠胃，芒硝软坚润燥、助大黄泻下，两药合用泻下攻积之力倍增，为大承气汤之核心组合。


## 四、配伍禁忌——十八反与十九畏


**十八反**（两药合用产生毒副作用）：甘草反甘遂、大戟、海藻、芫花；乌头反贝母、瓜蒌、半夏、白蔹、白及；藜芦反人参、沙参、丹参、玄参、苦参、细辛、芍药。歌诀："本草明言十八反，半蒌贝蔹及攻乌，藻戟遂芫俱战草，诸参辛芍叛藜芦。"


**十九畏**（两药合用减弱药效或产生不良反应）：硫黄畏朴硝，水银畏砒霜，狼毒畏密陀僧，巴豆畏牵牛子，丁香畏郁金，川乌/草乌畏犀角，牙硝畏三棱，官桂畏赤石脂，人参畏五灵脂。


**妊娠禁忌药**：禁用药包括巴豆、斑蝥、水蛭、虻虫、三棱、莪术、麝香等峻烈药物；慎用药包括桃仁、红花、大黄、附子、肉桂、枳实等活血化瘀、行气破滞及辛热类药物。
