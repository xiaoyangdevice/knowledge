# 阿司匹林（Aspirin）药物说明书

## 基本信息

- **通用名称**：阿司匹林（乙酰水杨酸）
- **英文名称**：Aspirin (Acetylsalicylic Acid)
- **药物分类**：非甾体抗炎药（NSAID）/ 抗血小板药
- **化学式**：C₉H₈O₄
- **分子量**：180.16 g/mol
- **CAS号**：50-78-2

## 药理作用

阿司匹林通过不可逆地抑制环氧化酶（COX）发挥作用：

1. **抗血小板作用（低剂量75-150mg）**：不可逆抑制血小板COX-1，减少血栓素A2（TXA2）的生成，抑制血小板聚集。由于血小板无细胞核，不能重新合成COX，因此抑制作用持续血小板的整个生命周期（7-10天）。

2. **解热镇痛抗炎作用（中高剂量300-1000mg）**：抑制前列腺素的合成，减少炎症介质的产生，发挥解热、镇痛和抗炎作用。

## 适应证

### 心血管适应证（低剂量）
- 急性冠脉综合征（ACS）的一级和二级预防
- 急性心肌梗死
- 经皮冠状动脉介入治疗（PCI）术后
- 缺血性卒中和短暂性脑缺血发作（TIA）的二级预防
- 外周动脉疾病
- 心房颤动患者不能使用口服抗凝药时的替代治疗

### 解热镇痛适应证（中高剂量）
- 发热
- 头痛、牙痛、肌肉痛
- 类风湿性关节炎
- 骨关节炎
- 强直性脊柱炎
- 痛风性关节炎急性发作

## 用法用量

| 适应证 | 剂量 | 频次 |
|--------|------|------|
| 抗血小板（长期预防） | 75-100mg | 每日一次 |
| 急性心肌梗死 | 首次300mg嚼服 | 之后75-100mg/日 |
| PCI术后 | 100mg | 每日一次 |
| 解热镇痛 | 300-600mg | 每4-6小时，最大4g/日 |
| 类风湿性关节炎 | 3-5g/日 | 分3-4次 |
| 川崎病 | 急性期30-50mg/kg/日 | 恢复期3-5mg/kg/日 |

## 不良反应

### 常见不良反应
- 胃肠道反应：恶心、呕吐、上腹部不适、胃灼热
- 消化道出血：黑便、呕血（发生率约1-2%/年）

### 少见不良反应
- 过敏反应：荨麻疹、血管性水肿、支气管痉挛（阿司匹林哮喘）
- 出血倾向：皮肤瘀斑、牙龈出血
- 肝肾功能损害
- Reye综合征（儿童病毒感染期间使用）

## 禁忌证

1. 对阿司匹林或其他水杨酸类药物过敏
2. 活动性消化道溃疡或出血
3. 严重肝肾功能不全
4. 出血性疾病（血友病、血小板减少症）
5. 妊娠晚期（第三个月）
6. 16岁以下儿童病毒感染（Reye综合征风险）

## 药物相互作用

- **华法林**：增加出血风险
- **布洛芬**：可能减弱阿司匹林的抗血小板作用
- **甲氨蝶呤**：减少其肾脏排泄，增加毒性
- **ACE抑制剂**：可能降低降压效果
- **丙磺舒**：减弱其促尿酸排泄作用
- **口服降糖药**：可能增强降糖作用

## 注意事项

1. 空腹时服用吸收更快，但为减少胃肠道刺激建议饭后服用
2. 肠溶片应整片吞服，不要咀嚼或压碎
3. 手术前7-10天应停用阿司匹林
4. 长期使用者应定期检查血常规和粪便潜血
5. 老年人应从低剂量开始
6. 哺乳期妇女使用需权衡利弊

## 贮藏

密封，在干燥处保存，温度不超过25°C。
