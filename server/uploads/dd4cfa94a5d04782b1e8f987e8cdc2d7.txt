奥美拉唑（Omeprazole）药物信息

一、基本信息

通用名：奥美拉唑
英文名：Omeprazole
商品名：洛赛克（Losec）、奥克（Aoke）等
药物分类：质子泵抑制剂（PPI）
化学式：C17H19N3O3S
分子量：345.42 g/mol
剂型：肠溶胶囊（10mg、20mg）、肠溶片、注射用粉剂（40mg）

二、作用机制

奥美拉唑是一种前药（prodrug），本身无活性。口服后在胃壁细胞分泌小管的酸性环境中（pH<4）转化为活性形式——次磺酰胺，与壁细胞H+/K+-ATP酶（质子泵）的半胱氨酸残基形成不可逆的共价键，使质子泵永久失活。

每次给药可抑制约70%的质子泵。由于新的质子泵持续合成（半衰期约50小时），需连续给药2-5天才能达到最大抑酸效果。停药后抑酸作用可持续2-3天。

奥美拉唑是目前最强效的抑酸药物之一，可将胃内pH升高至4以上长达16-18小时/天，使24小时胃酸分泌减少约80%。

三、适应证

1. 胃食管反流病（GERD）
   - 糜烂性食管炎的初始治疗和维持治疗
   - 非糜烂性反流病（NERD）的症状控制

2. 消化性溃疡
   - 胃溃疡：20mg/日，疗程6-8周
   - 十二指肠溃疡：20mg/日，疗程4周
   - NSAID相关溃疡的治疗和预防

3. 幽门螺杆菌根除治疗
   - 标准三联方案：奥美拉唑 + 克拉霉素 + 阿莫西林（或甲硝唑）
   - 铋剂四联方案：奥美拉唑 + 铋剂 + 两种抗生素
   - 剂量：20mg，每日2次，疗程14天

4. 卓-艾综合征（胃泌素瘤）
   - 起始剂量：60mg/日
   - 根据症状和胃酸分泌调整，可达120mg/日

5. 应激性溃疡的预防
   - ICU重症患者的预防性用药
   - 注射剂：40mg静脉滴注，每12小时一次

6. 上消化道出血
   - 大剂量方案：80mg静脉推注后，8mg/h持续泵入72小时

四、用法用量

口服：
- 成人常规剂量：20mg，每日1次，晨起空腹服用
- 重症或疗效不佳：20mg，每日2次或40mg每日1次
- 维持治疗：10-20mg/日

注射：
- 静脉注射/滴注：40mg，每12小时一次
- 上消化道出血：80mg负荷量后8mg/h持续泵入

特殊人群：
- 老年人：一般无需调整剂量
- 肝功能不全：每日剂量不超过20mg
- 肾功能不全：无需调整剂量
- 妊娠期：C类，权衡利弊后使用

给药提示：
- 餐前30-60分钟服用效果最佳
- 肠溶胶囊/片应整粒吞服，不可咀嚼或压碎
- 与食物同服不影响疗效但会延迟达峰时间

五、不良反应

常见（发生率1-10%）：
- 头痛（约7%）
- 腹泻（约3%）
- 腹痛（约3%）
- 恶心（约2%）
- 头晕、口干、便秘

长期使用的潜在风险（>1年）：
1. 低镁血症：长期使用可导致血镁降低，表现为肌肉痉挛、心律失常
2. 骨质疏松和骨折风险增加：尤其是髋部骨折，FDA已发出警告
3. 维生素B12吸收减少
4. 铁吸收减少
5. 难辨梭菌感染（CDI）风险增加
6. 社区获得性肺炎风险轻度增加
7. 肾脏损害：急性间质性肾炎、慢性肾病风险

六、药物相互作用

1. 氯吡格雷：奥美拉唑通过抑制CYP2C19降低氯吡格雷的活性代谢产物浓度，可能减弱抗血小板效果。建议改用泮托拉唑或雷贝拉唑。

2. 甲氨蝶呤：PPI可减少甲氨蝶呤的肾清除，增加毒性风险

3. 地高辛：胃酸pH升高可增加地高辛吸收，需监测血药浓度

4. 抗真菌药（酮康唑、伊曲康唑）：胃酸减少影响其吸收，降低疗效

5. 阿扎那韦等HIV蛋白酶抑制剂：PPI显著降低其吸收，不建议联用

6. 铁剂：胃酸减少影响铁的吸收

七、注意事项

1. 使用PPI前应排除胃癌等恶性疾病，因PPI可掩盖症状延误诊断
2. 尽可能使用最低有效剂量和最短疗程
3. 长期使用者应定期评估是否仍需继续治疗
4. 长期使用者建议监测血镁、维生素B12水平
5. 突然停药可能出现反跳性胃酸分泌增多，建议逐步减量
6. 肝功能不全者慎用，需减量
7. 可能影响尿蛋白定性试验结果

八、贮藏

遮光、密封，25°C以下保存。注射用粉剂溶解后应在4小时内使用。
